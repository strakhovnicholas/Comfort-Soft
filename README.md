# ComfortSoftTask

Приложение для поиска N-го минимального числа в XLSX файле.

## Инструкция по сборке и запуску кода

### Запуск с помощью Docker (рекомендуется)

Этот способ не требует установки Maven или JDK на вашей локальной машине. Требуется только Docker.

1.  **Убедитесь, что у вас установлен Docker Desktop.**

2.  **Скачайте исходный код:**
    ```bash
    git clone https://github.com/strakhovnicholas/Comfort-Soft.git
    cd Comfort-Soft # Переход в папку проекта
    ```

3.  **Соберите Docker-образ (включает в себя сборку Maven внутри контейнера):**
    ```bash
    docker build -t comfort-soft-app .
    ```
    (Замените `comfort-soft-app` на любое имя для вашего образа)

4.  **Убедитесь, что другие процессы не занимают порт. Если все хорошо - запустите контейнер:**
    ```bash
    docker run -p 8080:8080 comfort-soft-app
    ```

5.  **Откройте приложение:**
    Откройте браузер и перейдите по адресу `http://localhost:8080`.

После запуска приложения вы можете использовать его интерфейс для загрузки XLSX файла и поиска N-го минимального числа.

### Запуск с помощью Maven и Java (локально, опционально)

Этот способ требует предварительной установки JDK и Maven.

1.  **Убедитесь, что у вас установлены:**
    *   Java Development Kit (JDK) версии 21 (или совместимой с вашим проектом).
    *   Apache Maven.

2.  **Скачайте исходный код:**
    ```bash
    git clone https://github.com/strakhovnicholas/Comfort-Soft.git
    cd Comfort-Soft # Переход в папку проекта
    ```

3.  **Соберите проект:**
    ```bash
    mvn clean package
    ```
    В результате в папке `target` должен появиться JAR-файл (например, `ComfortSoftTask-0.0.1-SNAPSHOT.jar`).

4.  **Запустите приложение:**
    ```bash
    java -jar target/ComfortSoftTask-0.0.1-SNAPSHOT.jar
    ```
    (Замените `ComfortSoftTask-0.0.1-SNAPSHOT.jar` на точное имя вашего JAR-файла, если оно отличается)

5.  **Откройте приложение:**
    Откройте браузер и перейдите по адресу `http://localhost:8080`.
